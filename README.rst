============================
Stwitto - Twitter CLI client
============================


How to install :
----------------

::

 $ git clone https://github.com/surajnarwade/stwitto.git

 $ python setup.py install

Use case:
---------

::

 $ stwitto --tweet <tweet>

 $ stwitto --tweet <tweet> --image <image path>

 $ stwitto --help
